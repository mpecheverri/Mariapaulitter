# Node API Rest

## Clone the repo
```git clone https://github.com/mpecheverri/Mariapaulitter.git```

## install the app
```yarn```

## create .env file
```
HTTP_PORT=
LOG_ACCESS=
JWTKEY=
APIWEATHERKEY=
```

## run the app
```yarn start```

## Dev mode

### install nodemon globally
```yarn install --global nodemon```

### run the app
```yarn run dev```

### run tests
```yarn run test```
